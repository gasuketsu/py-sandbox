## 課題

```
#################################
S   #     #               #   # #
# # # ##### ########### # # # # #
# #   #     #     #     #   #   #
# ### # ####### # # #############
#   #   #     # # #       #     #
### ##### # # # # ####### # ### #
#   #   # # #   # #       #   # #
# ### # # ####### # ####### ### #
#     # #   #   # #   #     #   #
####### ### ### ### # # ##### ###
#       # #       # # #     #   #
# ####### # ### # ### ##### # ###
#   #     # #   # #   #   # #   #
### # ####### # # # # # # # ### #
#     #       # #   #   #   #   G
#################################

```

↑みたいな迷路をスタート(`S`) からゴール(`G`)まで辿る

### 制約条件

- スタートの位置必ず同じ(迷路の一番左上)で、目印として `S` が付いている
- ゴールの位置には目印として `G` がある
- ゴールにたどり着くまでの間に辿った道には`.`で印をつける。
- 迷路の形、大きさは必ずしも同じとは限らない

迷路そのものは2次元の文字の配列(リスト)で与えられるという想定でOK
